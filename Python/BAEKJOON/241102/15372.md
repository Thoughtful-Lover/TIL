# 문제 이해
### 15372. A Simple Problem
* 양의 정수 N을 입력 받았을 때
* 가장 작은 양의 정수 K의 값은?
  * K는 N의 제곱의 배수
### 입력
* 입력의 첫 줄은 테스트 케이스 수 T
* 각각의 테스트 케이스는 숫자 하나짜리 N개
* 1<=T<=200,000, 1<=N<=200,000
### 춢력
* 각 테스트 케이스 별, 정수 K를 출력
# 문제 구상
* 문제를 제대로 이해했는지 모르겠다.
* 그냥 제곱한 값 구하는거 아님?
# 문제 풀이
### 1차 시도
```python
# 테스트 케이스 수가 최대 20만개이니까 빠른 입력
import sys

# 테스트 케이스 수 입력
T = int(input())
# 테스트 케이스별 시행
for _ in range(T):
    # N을 입력 받고
    N = int(sys.stdin.readline())
    # N의 제곱의 배수 중 가장 작은 수는 그냥 N의 제곱
    print(N**2)
```
