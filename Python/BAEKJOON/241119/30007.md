# 문제 이해
### 30007. 라면 공식
* W = A*(X-1)+B
* W는 물의 양, A는 라면 계수, B는 기본 물의 양, X는 끓일 라면 수
### 입력
* 첫째 줄에 예찬이가 라면을 끓이는 횟수 N
* 둘째 줄부터 N+1번째 줄까지 A, B, X가 공백으로 구분되어
### 출력
* N개의 줄, 물의 양
### 제한
* 1<=N<=100
* 100<=A,B<=1,000 
* 1<=X<=100
# 문제 구상
* 위에 주어진 공식을 역산하면 된다.
# 문제 풀이
### 1차 시도
```python
N = int(input())
for _ in range(N):
    A, B, X = map(int, input().split())
    print(A*(X-1)+B)
```
* 31,120KB
* 36ms
