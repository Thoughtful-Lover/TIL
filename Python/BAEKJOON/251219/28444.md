# 문제 이해
### 28444. HI-ARC? (브론즈 5)
* HI-ARC 수식
  * H와 I의 곱이 첫 번째 항
  * A와 R과 C의 곱이 두 번째 항
  * 결과값은 첫 번째 항에서 두 번째 항을 뺀 값
### 입력
* 첫째 줄에 각 문자에 들어갈 값 H, I, A, R, C (0<=H,I,A,R,C<=100, H,I,A,R,C는 정수)가 공백을 사이에 두고 순서대로 주어짐
### 출력
* 수식을 계산한 결과값을 출력
# 문제 구상
* 사칙연산 문제
# 문제 풀이
### 1차 시도
```python
# H, I, A, R, C의 값을 각각 입력  받음
H, I, A, R, C = map(int, input().split())
# HI-ARC 의 값을 출력
print(H*I-A*R*C)
```
