# 문제 이해
### 24736. Football Scoring (브론즈 5)
* 득점을 하니 5가지 방법
  * Touchdown - 6 points
  * Field Goal - 3 points 
  * Safety - 2 points
  * After touchdown
    * 1 point (Field Goal or Safety) - Typically called the “Point after”
    * 2 points (touchdown) - Typically called the “Two-point conversion”
* 득점표가 주어졌을 때, 각각의 팀의 총 득점을 계산하라
### 입력
* 두 줄의 입력
  * 각각의 입력은 5개의 나뉜 입력값들, 음수가 아님
  * T, F, S, P, C
    * Touchdowns, Field goals, Safeties, Pints-after-touchdown and two-point Conversions
    * (0<=T<=10), (0<=F<=10), (0<=S<=10), (0<=(P+C)<=T)
* 첫 번째 입력 줄은 원정팀, 두 번째 입력 줄은 홈팀
### 출력
* 각각의 출력값이 하나의 줄에 빈 공백을 두고 출력
# 문제 구상
* 단순 사칙 연산 문제
# 문제 풀이
### 1차 시도
```python
# 각 팀의 점수를 입력 받음
visiting_team = list(map(int, input().split()))
home_team = list(map(int, input().split()))
# 각 팀의 득점 정보를 담은 2차원 배열 teams 와 각 팀의 최종 득점정보를 담을 scores 를 정의
teams = [visiting_team, home_team]
scores = [0]*2
# teams 배열을 순회하며 점수를 계산하고, 결과값을 scores 배열에 저장
for i in range(2):
    scores[i] = teams[i][0]*6 + teams[i][1]*3 + teams[i][2]*2 + teams[i][3]*1 + teams[i][4]*2
# 각각의 값을 출력
print(scores[0], scores[1])
```
