# 문제 풀이
### 11943. 파일 옮기기 (브론즈 4)
* 두 개의 바구니에 사과와 오렌지
  * 첫 번째 바구니에는 사과 A개, 오렌지 B개
  * 두 번째 바구니에는 사과 C개, 오렌지 D개
* 한 바구니에 있는 과일 하나를 집어서 다른 바구니로 옮김
* 한 바구니에는 사과만 있게, 다른 쪽에는 오렌지만 있게
* 앞서 말한 조건을 만족하도록 과일을 옮길 때, 과일을 옮기는 최소 횟수를 구하는 프로그램
### 입력
* 첫 번째 줄에는 첫번째 바구니에 있는 사과와 오렌지의 수 A, B (0<=A,B<=1,000)
* 두 번째 줄에는 두 번째 바구니에 있는 사과와 오렌지의 수 C,D (0<=C,D<=1,000)
### 출력
* 사과와 오렌지를 옮기는 최소 횟수
# 문제 구상
* 각각의 바구니에 사과와 오렌지만 있기 위해서는 한 쪽에서 사과를 옮기면 다른 쪽에서 바나나를 옮겨야 한다
* 즉, A+D, B+C 중 낮은 값을 고르면 됨
# 문제 풀이
### 1차 시도
```python
# A, B, C, D를 각각 입력 받음
A, B = map(int, input().split())
C, D = map(int, input().split())
# 첫 번째 바구니에서 사과, 두 번째 바구니에서 오렌지를 고르거나
# 첫 번째 바구니에서 오렌지, 두 번째 바구니에서 사과
# 두 가지 경우 중 더 낮은 값을 출력
print(min(A+D, B+C))
```
