# 문제 이해
### 32384. 사랑은 고려대입니다 (브론즈 5)
### 입력
* 첫 번째 줄에 정수 N(1<=N<=10)이 주어짐
### 출력
* 첫 번째 줄에 ```LoveisKoreaUniversity```를 공백으로 구분하여 N번 출력
# 문제 구상
* 단순 구현 문제
# 문제 풀이
### 1차 시도
```python
# N을 입력 받음
N = int(input())
# N번 만큼 반복하되
for _ in range(N):
    # 다음 출력 시 공백을 두고 출력하도록 설정
    print('LoveisKoreaUniversity', end=' ')
```
