# 문제 이해
### 5988. 홀수일까 짝수일까 (브론즈 3)
* N개의 정수가 주어지면 홀수인지 짝수인지 출력하는 프로그램
### 입력
* 첫 번째 줄에 숫자의 개수 N (1<=N<=100)
* 두 번째 줄부터 N+1번째 줄에 걸쳐 홀수인지 짝수인지 확인할 정수 K (1<=K<=10^60)
### 출력
* N개의 줄에 걸쳐 한 줄씩 정수 K가 홀수라면 'odd', 짝수라면 'even'을 출력
# 문제 구상
* 반복문 + 조건문
# 문제 풀이
### 1차 시도
```python
# N번 만큼 시행
N = int(input())
for _ in range(N):
    # 수를 입력 받고
    num = int(input())
    # 2로 나눈 값에 나머지가 있으면 홀수
    if num%2:
        print('odd')
    # 아니라면 짝수
    else:
        print('even')
```
