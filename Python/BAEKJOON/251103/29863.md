# 문제 이해
### 29863. Arno's Sleep Schedule (브론즈 5)
* Arno가 얼마나 잤는지 시간을 계산할 수 있는 프로그램을 작성하라
### 입력
* 두 줄로 입력
* 첫 번째 줄은 20-23, 또는 0-3 이 입력값으로 주어짐. 잠든 시간
* 두 번째 줄은 5-10 이 주어짐. 깨어난 시간
* 모든 입력 값은 integer 값으로 24시간 형식으로 주어짐
### 출력
* 하나의 integer를 출력
* 잠든 시간에서 깨어난 시간까지의 시간을 출력
* 도중에 Arno는 깨지 않는다고 가정
* Arno는 24시간 이상 자지 않음
# 문제 구상
* 조건문으로 0시 이전에 잠든 경우와 이후에 잠든 경우를 나눠서 계산해야할 것 같음
# 문제 풀이
### 1차 시도
```python
# 잠든 시간과 일어난 시간을 각각 입력 받고
go_to_bed = int(input())
get_up = int(input())

# 일어난 시간에서 잠든 시간을 빼줌
sleep = get_up-go_to_bed
# 단 이때, 해당 값이 0보다 작은 경우 전날에 잠들어서 다음날 깬 경우이므로
if sleep < 0:
    # 24를 추가해 줌
    sleep += 24
# 잠잔 시간을 출력
print(sleep)
```
