# 문제 이해
### 32951. AI 선도대학 (브론즈 5)
* 2024년은 AI 선도대학 원년
* 그로부터 몇 년 후, N년이 되었다.
* 연도 N이 입력으로 주어질 때, AI 선도대학 원년으로부터 몇 년이 지났는지 출력
### 입력
* 첫 번째 줄에 궁금한 연도 N이 주어짐 (2024<=N<=9999)
### 출력
* 첫 번째 줄에 N년이 2024년으로부터 몇 년이나 지났는지 출력
# 문제 구상
* 주어진 연도에서 2024년을 빼주면 된다.
# 문제 풀이
### 1차 시도
```python
N = int(input())
print(N-2024)
```
* 32,412 KB
* 40 ms
