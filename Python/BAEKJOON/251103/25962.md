# 문제 이해
### 15962. 새로운 시작 (브론즈 5)
* 지금 시간이 UTC+0(세계 표준시)을 기준으로 무슨 날짜인지 출력해주는 프로그램을 작성
### 입력
* 입력은 없다.
### 출력
* UTC+0(세계 표준시)을 기준으로 나타냈을 때의 연도, 월, 일을 한 줄에 하나씩 순서대로 출력
# 문제 구상
* 현재 시간을 출력하는 메서드를 활용해보자
# 문제 풀이
### 1차 시도
```python
# 날짜를 활용하기 위해 datetime 을 import
import datetime

# 오늘 날짜를 UTC+0 로 가져오기
# 한국은 UTC+9 이므로 9시간을 빼주고
today = datetime.date.today()-datetime.timedelta(hours=9)
# 연, 월, 일을 각각 출력
print(today.year)
print(today.month)
print(today.day)
```
