# 문제 이해
### 5717. 상근이의 친구들 (브론즈 5)
* 상근이의 남자 친구의 수와 여자 친구의 수가 주어질 때, 친구는 총 몇 몇인지 구하는 프로그램을 작성
### 입력
* 입력은 여러 개의 테스트 케이스로 구성
* 각 테스트 케이스는 두 정수 M과 F로 이루어져 있음
* 각각은 상근이의 남자 친구의 수와 여자 친구의 수 (1<=M, F<=5)
* 입력의 마지막 줄에는 0 이 두개 주어짐
### 출력
* 각 테스트 케이스마다 상근이의 친구의 수를 출력
# 문제구상
* 단순 사칙연산 문제
* 종료 조건(마지막 줄에 0이 두 개 주어질 때)만 잘 살피면 됨
# 문제 풀이
### 1차 시도
```python
# 반복
while True:
    # 남자인 친구의 수와 여자인 친구의 수를 각각 M,F로 입력 받음
    M, F = map(int, input().split())
    # 종료 조건: M과 F가 모두 0인 경우
        # 처음에 'if not M'으로 조건을 설정했지만, 생각해보니 남자인 친구가 0인 경우도 존재할 수 있음
    if not M and not F:
        break
    # 만약 위의 종료 조건이 만족되지 못할 경우, M과 F의 값의 합을 출력
    print(M+F)
```
